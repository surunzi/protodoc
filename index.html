TaskMVC

  • Add Task
    P
    http://www.taskmvc.com/task/add

    Add task to list.

    You don't have to send the id of task. It will be generated automatically by server, same as createTime and complete.

    PARAMS

    Name Type Description
    title string Task title
    detail string Task detail

    RETURN

    Name Type Description
    id number Id of the new task

    Open developer tool to check details

  • Delete Task
    G
    http://www.taskmvc.com/task/add

    RETURN

    Name Type Description
    status number Whether is successful, 0 means ok
  • Delete all Tasks
    G
    http://www.taskmvc.com/tasks/delete

    RETURN

    Name Type Description
    status string Successful or not
  • Edit Task
    G
    http://www.taskmvc.com/task/edit

    PARAMS

    Name Type Description
    id number Task id
    title string Task title
    detail string Task detail
    complete number 1 means done

    RETURN

    Name Type Description
    status number save or not

    Open developer tool to check details

  • Get Tasks
    G
    http://www.taskmvc.com/tasks/get

    Get all Tasks. Each task contains the following content:

    1. Id
    2. Title
    3. Detail
    4. CreateTime
    5. Complete

    RETURN

    Name Type Description
    tasks array List of tasks
